# Homework Week 4  
## Alyssa's Baking Cost Calculator

### Description

This program outputs three different cases based on the input given from the user. It can calculate the total cost of a recipe 
based on what dessert you are making, as well as the quantity needed. It can also transfer your baking report details to the 
report.txt file and give you an idea of how efficiently you are baking, all while in a loop that won't close until the user 
chooses to exit it. 

### Inputs

1. Recipe Name?
2. Quantity?
3. Cost per dessert? 

### Outputs

1. Summary of recipe & total cost (also outputs to report.txt file).
2. Shows baking report in the program
3. Shows baking efficiency.
4. Exit
